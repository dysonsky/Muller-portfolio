            <h2>üìö Day 1: Introduction to HTML</h2>
            <h3>What is HTML?</h3>
            <p>HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags.</p>
            
            <h3>Basic HTML Structure</h3>
            <div class="code-example">
&lt;!DOCTYPE html&gt;<br>
&lt;html&gt;<br>
&lt;head&gt;<br>
    &lt;title&gt;My First Page&lt;/title&gt;<br>
&lt;/head&gt;<br>
&lt;body&gt;<br>
    &lt;h1&gt;Hello World!&lt;/h1&gt;<br>
&lt;/body&gt;<br>
&lt;/html&gt;
            </div>

            <h3>Key Concepts:</h3>
            <ul>
                <li>Every HTML document starts with &lt;!DOCTYPE html&gt;</li>
                <li>&lt;html&gt; is the root element</li>
                <li>&lt;head&gt; contains meta information</li>
                <li>&lt;body&gt; contains the visible content</li>
            </ul>
        </div>

        <div class="day-section">
            <h2>üìö Day 2: Text Elements and Headings</h2>
            <h3>Headings</h3>
            <p>HTML has 6 levels of headings from &lt;h1&gt; to &lt;h6&gt;, where h1 is the largest and h6 is the smallest.</p>
            
            <div class="code-example">
&lt;h1&gt;Main Heading&lt;/h1&gt;<br>
&lt;h2&gt;Sub Heading&lt;/h2&gt;<br>
&lt;h3&gt;Smaller Heading&lt;/h3&gt;<br>
&lt;p&gt;This is a paragraph.&lt;/p&gt;<br>
&lt;p&gt;Text can be &lt;strong&gt;bold&lt;/strong&gt; or &lt;em&gt;italic&lt;/em&gt;.&lt;/p&gt;
            </div>

            <h3>Other Text Elements:</h3>
            <ul>
                <li>&lt;p&gt; - Paragraph</li>
                <li>&lt;strong&gt; - Bold text (important)</li>
                <li>&lt;em&gt; - Italic text (emphasis)</li>
                <li>&lt;br&gt; - Line break</li>
                <li>&lt;hr&gt; - Horizontal rule</li>
            </ul>
        </div>

        <div class="day-section">
            <h2>üìö Day 3: Lists and Links</h2>
            <h3>Lists</h3>
            <div class="code-example">
&lt;!-- Unordered List --&gt;<br>
&lt;ul&gt;<br>
    &lt;li&gt;Item 1&lt;/li&gt;<br>
    &lt;li&gt;Item 2&lt;/li&gt;<br>
&lt;/ul&gt;<br><br>
&lt;!-- Ordered List --&gt;<br>
&lt;ol&gt;<br>
    &lt;li&gt;First&lt;/li&gt;<br>
    &lt;li&gt;Second&lt;/li&gt;<br>
&lt;/ol&gt;
            </div>

            <h3>Links</h3>
            <div class="code-example">
&lt;a href="https://example.com"&gt;Visit Example&lt;/a&gt;<br>
&lt;a href="https://example.com" target="_blank"&gt;Open in New Tab&lt;/a&gt;
            </div>
        </div>

        <div class="day-section">
            <h2>üìö Day 4: Images and Media</h2>
            <h3>Adding Images</h3>
            <div class="code-example">
&lt;img src="image.jpg" alt="Description of image"&gt;<br>
&lt;img src="photo.jpg" alt="Photo" width="300" height="200"&gt;
            </div>

            <h3>Key Attributes:</h3>
            <ul>
                <li>src - Source of the image</li>
                <li>alt - Alternative text for accessibility</li>
                <li>width/height - Dimensions in pixels</li>
            </ul>
        </div>

        <div class="day-section">
            <h2>üìö Day 5: Tables</h2>
            <h3>Creating Tables</h3>
            <div class="code-example">
&lt;table border="1"&gt;<br>
    &lt;tr&gt;<br>
        &lt;th&gt;Name&lt;/th&gt;<br>
        &lt;th&gt;Age&lt;/th&gt;<br>
    &lt;/tr&gt;<br>
    &lt;tr&gt;<br>
        &lt;td&gt;John&lt;/td&gt;<br>
        &lt;td&gt;25&lt;/td&gt;<br>
    &lt;/tr&gt;<br>
&lt;/table&gt;
            </div>

            <h3>Table Elements:</h3>
            <ul>
                <li>&lt;table&gt; - Table container</li>
                <li>&lt;tr&gt; - Table row</li>
                <li>&lt;th&gt; - Table header</li>
                <li>&lt;td&gt; - Table data cell</li>
            </ul>
        </div>

        <div class="day-section">
            <h2>üìö Day 6: Forms - Part 1</h2>
            <h3>Basic Form Structure</h3>
            <div class="code-example">
&lt;form action="/submit" method="post"&gt;<br>
    &lt;label for="name"&gt;Name:&lt;/label&gt;<br>
    &lt;input type="text" id="name" name="name"&gt;<br><br>
    &lt;label for="email"&gt;Email:&lt;/label&gt;<br>
    &lt;input type="email" id="email" name="email"&gt;<br><br>
    &lt;input type="submit" value="Submit"&gt;<br>
&lt;/form&gt;
            </div>

            <h3>Input Types:</h3>
            <ul>
                <li>text - Single line text</li>
                <li>email - Email address</li>
                <li>password - Password field</li>
                <li>number - Numeric input</li>
                <li>submit - Submit button</li>
            </ul>
        </div>

        <div class="day-section">
            <h2>üìö Day 7: Forms - Part 2 & Semantic HTML</h2>
            <h3>More Form Elements</h3>
            <div class="code-example">
&lt;textarea rows="4" cols="50"&gt;Enter text here&lt;/textarea&gt;<br><br>
&lt;select name="country"&gt;<br>
    &lt;option value="ke"&gt;Kenya&lt;/option&gt;<br>
    &lt;option value="us"&gt;USA&lt;/option&gt;<br>
&lt;/select&gt;<br><br>
&lt;input type="radio" name="gender" value="male"&gt; Male<br>
&lt;input type="radio" name="gender" value="female"&gt; Female<br><br>
&lt;input type="checkbox" name="agree"&gt; I agree
            </div>

            <h3>Semantic HTML Elements:</h3>
            <ul>
                <li>&lt;header&gt; - Header section</li>
                <li>&lt;nav&gt; - Navigation links</li>
                <li>&lt;main&gt; - Main content</li>
                <li>&lt;footer&gt; - Footer section</li>
                <li>&lt;article&gt; - Independent content</li>
                <li>&lt;section&gt; - Thematic grouping</li>
            </ul>
        </div>
